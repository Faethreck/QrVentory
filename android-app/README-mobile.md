# QrVentory Android (scanner)

Quick Android client to scan QrVentory QR codes and show their data.

## What it does
- Uses the device camera to scan QR codes.
- Parses the JSON payload generated by the desktop app (Nombre, NoSerie, Categoria, Ubicacion).
- Shows the last scans in a simple list; you can clear the history.

## Project layout
- `android-app/app/src/main/java/com/qrventory/mobile/MainActivity.kt` — scanning + UI logic.
- `android-app/app/src/main/java/com/qrventory/mobile/ScannedItemAdapter.kt` — list adapter.
- `android-app/app/src/main/java/com/qrventory/mobile/ScannedItem.kt` — data model.
- `android-app/app/src/main/res/layout/activity_main.xml` — main screen layout.
- `android-app/app/src/main/res/layout/item_scan.xml` — row layout for history cards.
- `android-app/app/src/main/AndroidManifest.xml` — camera permission + launcher activity.

## Build/run
1) Open `android-app` in Android Studio (Electric Eel or newer) and let it install the Android Gradle Plugin 8.5+ and Kotlin 1.9+.
2) Plug in an Android device or start an emulator (API 24+).
3) Press **Run**. The app uses `com.journeyapps:zxing-android-embedded` for scanning; no extra setup needed beyond camera permission.

## Usage
- Tap **Escanear QR** and point at a QrVentory QR.
- The decoded item appears in the list with serial/name/category/location.
- Tap **Limpiar** to reset the history.

## Notes
- If a QR contains non-JSON content, the raw text is shown.
- Payload keys expected: `Nombre`, `NoSerie`, `Categoria`, `Ubicacion` (matches the desktop QR payload).
